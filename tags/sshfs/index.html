Linux环境下建立SSHFS挂靠的方法

Linux环境建立SSHFS共享的方法

客户端安装sshfs及fuse

  • CentOS环境安装命令：sudo yum install sshfs fuse
  • Ubuntu环境安装命令：sudo apt-get install sshfs fuse

    手动安装sshfs-fuse的方法：

  • tar -xzvf sshfs-fuse-2.4.tar.gz
  • cd sshfs-fuse-2.4
  • ./configure
  • make -j
  • sudo make install

配置ssh自动登陆

  • 在客户端使用普通用户生成密钥对,执行命令：ssh-keygen -d
  • 通过客户端，将公钥上传至服务端,
  • 执行命令：ssh-copy-id -i ~/.ssh/id_dsa.pub 服务器IP

建立sshfs共享

  • ssh username@服务器IP:/服务器中要共享的文件目录/ /共享到本地的文件目录/
